The Solution

We are creating a single, universal client for all platforms, leveraging the Web UI and Kotlin Multiplatform (KMP). This unified codebase allows us to support multiple platforms efficiently while maintaining native performance. Furthermore, it integrates robust offline capabilities directly into this single core.

Goodbye multiple clients, welcome Pega One Client!
